# FoodDelivery-Idea-MvRx
This is sample project have main page screen where you can see promotions, list of food items.
You can add food item to cart by tapping the price. FAB cart will update accordingly.
This simple project, It is build with MvRx, RXJava and Kotlin. This is experimental project used for learning.
If you want to take it as reference, please feel free to take and use it or learn it.
